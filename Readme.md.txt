Aplicación con PyQt5 para cargar imágenes y generar metadatos en formato JSON con un diseño moderno tipo página web.
@LuisTM207 Y @Adrianrivas117
Características

-  Vista previa de imágenes con diseño moderno
-  Formulario completo para metadatos (título, autor, descripción, etiquetas)
-  Guardado automático en JSON
-  Visualización de todas las imágenes guardadas
-  Exportación de metadatos a archivo JSON
-  Diseño tipo página web con gradientes y estilos modernos
-  Información detallada de archivos (tamaño, dimensiones)
